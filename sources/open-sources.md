# Open Source Notes

This repository is a curated learning-material layer. It does not mirror large corpora.

## Tatoeba

- URL: https://tatoeba.org/en/downloads
- License note: Tatoeba download page says its files are released under CC BY 2.0 FR, and part of the sentences are also available under CC0 1.0.
- Use in this repo: small curated inspiration for sentence patterns; no bulk mirror.

## Open English WordNet

- URL: https://github.com/globalwordnet/english-wordnet
- License note: project README says Open English WordNet is released under CC-BY 4.0.
- Use in this repo: vocabulary relationship inspiration, word families, synonym/near-synonym grouping.

## Princeton WordNet

- URL: https://wordnet.princeton.edu/license-and-commercial-use
- License note: official page says WordNet may be used in commercial applications according to its license, and permission is granted to use/copy/modify/distribute for any purpose without fee or royalty if license notices are preserved.
- Use in this repo: word sense/reference checking.
